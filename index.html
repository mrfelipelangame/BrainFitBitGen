Learn the Soroban Methods


Simple Method

Only direct addition and subtraction. Result must be 0–9.

Allowed sums: 1+1,1+2,1+3,1+5,2+5,3+5,4+5

Small Friend

Use bead5 with compensation. Result ≤ 9. Subtraction only if total ≥ number.

Example: 3+2 → +5−3=5, 5−1 → +5−1=4

Big Friend

Use 10 complement / borrow. Result ≥ 10. Always valid operation (total ≥ number).

Example: 8+7=15, 35−7=28

Mixed Method

Combines Small and Big Friend techniques as needed.

Example: 7+6 → +10−4=13
